# 🎓 Student Result Analyzer (C++)

Student Result Analyzer is a simple academic system built using **C++ 💻** that processes student marks and generates a complete result report.

The system calculates **grades, grade points, total grade points, and average grade point** for multiple subjects using a predefined grading system with exception handling.

---

# ✨ Features

📊 Multi-Subject Result Processing  
- Handles 9 subject codes  
- Accepts marks for each subject  

🎯 Grade Calculation  
- Grades: S, A+, A, B+, B, C+, C, U  

📈 Grade Point System  
- Converts grades to points  
- Calculates total  

🧮 Average Grade Point  
- Computes GPA  

⚠ Exception Handling  
- Prevents invalid input  

🧾 Student Report  
- Name, Roll, Course, Semester, Year  
- Subject-wise result

# 🧠 How It Works

Input → Student details + marks  

Grades:  
91–100 → S  
81–90 → A+  
71–80 → A  
66–70 → B+  
61–65 → B  
56–60 → C+  
50–55 → C  
<50 → U  

Points:  
S=10, A+=9, A=8, B+=7, B=6.5, C+=6, C=5, U=0  

Output:  
Total + Average Grade Points
